## Getting Started

#Front-end Start

First, run the development server:

```bash
cd front-end
npm run dev
# or
front-end
yarn dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

#Back-end Start

```bash
cd back-end
yarn develop
# or 
cd back-end
npm run develop

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
